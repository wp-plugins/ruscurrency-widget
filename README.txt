== Description ==

Show the exchange rate of The Central Bank of the Russian Federation.

**Supported Languages:**

* RU Russian (default)

**Demo:**

[http://vexsell.ru/](http://vexsell.ru/)

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To add the RuCurrency widget to the sidebar go to 'Appearance->Widgets', and add the RuCurrency to your blog.
